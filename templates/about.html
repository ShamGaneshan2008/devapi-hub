What is DevAPI Hub?

DevAPI Hub is an open developer marketplace where anyone can register, get a unique API key, and immediately start publishing or consuming REST APIs. Whether you've built a weather service, a finance data feed, or an AI-powered endpoint — you can list it here, share it with the community, and get feedback through ratings and comments.

Every registered user gets a personal API key that authenticates requests to protected platform APIs. The built-in test console lets you fire live HTTP requests and inspect responses, headers, and timing — right from your browser.

Platform Features

🔑

API Key Management

Every user gets a unique dah_ prefixed API key on registration. Regenerate it anytime from your dashboard — old keys invalidate instantly.

🚀

Publish APIs

List any REST API endpoint with a name, description, and category. Edit or remove your listings anytime. Duplicate endpoints are rejected automatically.

⚡

Live Test Console

Fire real GET requests to any endpoint and see the JSON response, HTTP status code, response time in milliseconds, and key response headers — all in one view.

🛡️

Rate Limiting

All protected API calls are rate-limited to 30 requests per minute per key. Responses include X-RateLimit-* headers so you always know where you stand.

⭐

Ratings & Comments

Community members can rate APIs 1–5 stars and leave comments. Ratings are averaged across all votes. Admins can moderate and delete comments.

👑

Admin Controls

The first registered user becomes admin. Admins get a gold badge, access to the control panel, request logs, user management, and full comment moderation.

Built-in Protected APIs

Weather API

GET /api/weather?key=…

Returns temperature, humidity, wind, UV index for London.

Joke API

GET /api/joke?key=…

Returns a random developer joke. A new one every call.

Status API

GET /api/status?key=…

Returns your usage count, rate limit remaining, and window info.

All APIs also accept your key via X-API-Key header instead of query param.
